pub fn build(self) -> Result<&'a mut TCTransferFrame, BuildError>

Consumes the builder and constructs the TCTransferFrame header.

This method is only available after all required fields (scid, vcid, buffer, and data_field_len) have been set.

Returns a mutable view of the frame, ready for the user to write data into its data field.
