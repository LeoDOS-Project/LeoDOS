Struct TCTransferFrame 

Source 
#[repr(C, packed(1))]
pub struct TCTransferFrame { /* private fields */ }
Expand description

A zero-copy view over a TC Transfer Frame in a raw byte buffer.

This struct can be created via TCTransferFrame::builder() and provides access to the frame header and its data field, which typically contains one or more SpacePackets.
