# LeoDOS

LEO Distributed Operating System -- a communication stack and distributed computing platform for LEO satellite constellations.

- **[SpaceCoMP](spacecomp/overview)** -- distributed computation across constellations
- **[Protocols](protocols/stack)** -- communication stack from physical to transport layer
- **[Simulation](simulation/simulator)** -- constellation simulator with NOS3 integration
- **[ColonyOS](colonyos/integration)** -- external job orchestration
