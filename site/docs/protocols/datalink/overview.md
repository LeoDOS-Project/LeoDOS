# Overview

The data link layer imposes structure on the data passing between
the network layer above and the coding layer below: frame
boundaries, addressing, multiplexing, security, and optional
per-hop reliability. It operates on individual point-to-point
links — each ISL and each ground link has its own independent
data link layer instance.

- [cFE Headers](cfe-headers/overview) — mission-specific secondary headers for TM and TC
- [Packet Protocol](packet/overview) — CCSDS Space Packet Protocol (SPP)
- [Transfer Frame](transfer-frame/overview) — frame protocols for point-to-point links
- [Security](security/overview) — Space Data Link Security (SDLS)
- [Reliability](reliability/overview) — COP-1 per-hop reliable delivery
