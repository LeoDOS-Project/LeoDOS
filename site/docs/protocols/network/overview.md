# Overview

The network layer provides addressing and multi-hop forwarding.
Below this layer, communication is point-to-point between
neighbours. Above it, any node can send to any other node in the
constellation.

- [Routing](routing) --- torus mesh routing with physics-aware algorithms
- [Point to Point](point-to-point) --- trivial forwarding for single-link setups
- [Routing](routing) --- detailed ISL routing protocol specification
- [Gossip](gossip) --- broadcast message dissemination
